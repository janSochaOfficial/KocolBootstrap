Jak zacząć korzystać z BootStrap'a?

Aby zacząć korzystać z BootStrap'a należy pobrać bibliotekę css i js ze strony https://getbootstrap.com/docs/5.0/getting-started/download/ po czym dodać odpowiednie impotry do kodu HTML

Można też pominąć krok pobierania i zaimportować BootStrapa prosto ze strony https://getbootstrap.com/docs/5.0/getting-started/introduction/
